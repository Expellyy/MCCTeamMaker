# MCCTeamMaker v1.0

## How to Use:
1. Extract this ZIP file (right click on the zip and press "Extract all" on the dropdown, or left click and click "Extract all" near the top of your screen".
2. Open the extracted folder by double-clicking. Do not take anything out of the folder.
3. Double-click `main.exe` to run the program.

## System Requirements:
- Windows 10/11 (64-bit)
- No installation required (:D)

## Notes:
- Keep `Fonts/` and `teams/` in the same folder as `main.exe`.
- The output file (saved as .png, title of your choosing) will be saved in the same directory.
